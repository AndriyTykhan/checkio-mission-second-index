どうやって解けばいいかわかりません。
文字列の中から、探している文字が最初に現れる箇所を探しましょう。見つかったら、その場所に移動して、もう一度探しましょう。それが二番目です。
何から始めればいいんでしょうか？
最初に、探している文字が少なくとも二回出現するかを確かめましょう。 出現しないなら、Noneを返す必要があります。 str.countメソッドが役に立つでしょう。
どんなコードができて、どのように動くのでしょうか？
最初はこのようになるでしょう: 
def second_index(text, symbol):
    if text.count(symbol) < 2: 
        return None
例えば、入力が'some text'で、探したい文字が'e'なら、Noneを返しません。 しかし、探したい文字を's'にした場合、'some text'.count ('s') == 1 で、かつ1 < 2を満たすので、Noneを返すでしょう。
次はどうすればいいでしょう？
探している文字が二回現れることがわかったので、次は最初に現れる場所を見つけましょう。 find メソッドが役に立ちます。 これは文字列から探している文字が最初に現れるインデックスを返します。例えば:
text = 'abcdec'
symbol = 'c'
first = text.find(symbol, 0)
ここで、括弧の中の0は省略可能です。この値は、関数が何番目の文字から検索をするかを表しており、0は文字列の一番始めを意味します。 この例では、'abcdec'の中で最初に現れる'c'のインデックスは2なので、'first'は2になります。
つまり…？
こうなるでしょう。 
def second_index(text, symbol):
    if text.count(symbol) < 2: 
        return None
    first = text.find(symbol)
つまり、もしtextに探している文字が2回未満しか現れなければ、Noneを返します。 もし2回以上現れるなら、文字列の中で最初に現れるインデックスを見つけます。
でも、二回目に現れる場所が必要なんですよね？
そうです。つまり、検索は最初からではなく、最初に探している文字が見つかった場所から検索しなければなりません。 この場所は'first'で見つけています。要するに、findで検索を開始する場所を0からfirst+1にすればよいのです。
まだわかりません！
最終的にはこうなります: 
def second_index(text, symbol):
    if text.count(symbol) < 2: 
        return None
    first = text.find(symbol)
    return text.find(symbol, first + 1)
例えばこの関数で 'a' を "I'm learning Python and nothing else" の中から探すとき、20を返します。 このコードによると、最初に 'a' が現れるのインデックスは6で (first = 6)、 その次 (first + 1) から検索を開始し、次に 'a' が現れるインデックスを20と決めることができます。
